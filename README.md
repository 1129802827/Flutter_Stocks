## 重大修改
因为原有新闻数据为爬取华尔街见闻全球资讯，由于爬虫失效，腾讯新闻也有限制，现更改为网易新闻数据

# 子不语股票
#### "子不语股票" 支持财经新闻阅读、实时大盘指数、实时沪深行情、k线查看、分享、登录、网页查看、侧边栏等功能



## 项目
项目使用Flutter进行开发，同时支持Andriod与iOS.
